## Notebooks

This folder contains Jupyter notebooks used for analysis, modeling, and validation in the RetailPulse project.

### Purpose
- Explore and understand the retail dataset
- Develop analytical and statistical models
- Validate assumptions and results before dashboarding

### Contents
- Sales forecasting using SARIMA
- Customer analysis and segmentation (RFM)

### Notes
- Notebooks are executed from top to bottom for reproducibility
- Modeling approaches were evaluated and refined during analysis
- Final insights are surfaced through SQL views and Power BI dashboards
